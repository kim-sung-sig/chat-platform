# AGENT_COMMANDS.md

Example Agent Command Collection

These are illustrative commands you can use when directing the agent.

- /plan: propose a plan and confirm before large changes
- /review: review recent changes with findings and risks
- /qa: create or extend test cases for a change
- /refactor: improve structure without changing behavior
- /explain: explain the current code or design
- /patch: implement a focused change
- /perf: analyze performance hotspots and suggest fixes
- /security: review for security risks
- /docs: update or draft documentation

Usage examples
- /plan Add caching for auth token lookups
- /review Review the login flow changes
- /qa Add unit tests for password policy
- /refactor Simplify user session handling
